# Agent Plans

This folder stores implementation plans for multi-step work. Use Cursor's Plan Mode (Shift+Tab) or create plans manually.

## Usage

1. Copy `_template.md` and rename for your task (e.g. `feature-auth.md`)
2. Fill in the goal, steps, and verification criteria
3. Reference the plan when working with the agent
4. Update checkboxes as you complete steps

## Naming

- `feature-*` — New features
- `refactor-*` — Refactoring work
- `fix-*` — Bug fix plans
- `chore-*` — Maintenance tasks
